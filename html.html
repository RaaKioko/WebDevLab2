Hypertext Markup Language (HTML)

HTML is the standard markup language for creating web pages. It uses elements (tags) to structure content — headings, paragraphs, links, images, forms, etc. The current version is HTML5.

Role in Technology:

HTML is the skeleton of every web page. Browsers parse HTML to understand the structure and meaning of content. CSS then styles it and JavaScript makes it interactive.
