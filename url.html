Uniform Resource Locators (URLs)

A URL is a standardized address used to locate resources on the Internet. Every web page, image, video, or API endpoint has a unique URL.

Role in Technology:

URLs provide a human-readable way to identify and access any resource on the Web. They tell the browser which protocol to use, which server to contact, and which specific resource to retrieve.
